# ProgressBar - AngularJS

- Built with AngularJS
- Linted with JSHint
- Pre compiled with SASS
- Bundled and minified js/css files with webpack

To run
<pre>npm start</pre>
or
<pre>node server.js</pre>

To Bundle - For Production Quality / Minified
<pre>webpack -p</pre>

To Bundle - Unminified
<pre>webpack</pre>
